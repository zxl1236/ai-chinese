# API密钥配置说明

## 在线DeepSeek API配置
1. 访问 https://platform.deepseek.com/
2. 注册账号并获取API密钥
3. 设置环境变量: set DEEPSEEK_API_KEY=your_api_key

## 阿里云Qwen API配置  
1. 访问 https://dashscope.aliyun.com/
2. 注册账号并获取API密钥
3. 设置环境变量: set QWEN_API_KEY=your_api_key

## Windows PowerShell设置环境变量:
$env:DEEPSEEK_API_KEY = "your_deepseek_key"
$env:QWEN_API_KEY = "your_qwen_key"

## Windows CMD设置环境变量:
set DEEPSEEK_API_KEY=your_deepseek_key
set QWEN_API_KEY=your_qwen_key

## 重启后端服务以使环境变量生效
cd backend
python app_refactored.py
